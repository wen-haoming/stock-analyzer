# 📈 AI 股票分析工具 (StockAnalyzer)

基于 Electron + React + AI 的股票分析桌面应用

## ✨ 功能特性

| 功能 | 描述 |
|------|------|
| 当日走势分析 | 分析当日 K 线，结合新闻给出影响因素权重 |
| 区间涨跌分析 | 选择日期区间，分析涨跌原因及归因 |
| 未来走势预测 | 基于历史数据预测 T+1/T+3/T+5 走势 |
| 思维链显示 | 可视化 AI 推理过程 |

## 🛠 技术栈

| 层级 | 技术 |
|------|------|
| 桌面框架 | Electron |
| 前端 | React + Ant Design |
| 样式 | UnoCSS |
| 构建 | Vite 8 |
| K线图 | lightweight-charts |
| AI | MiniMax 2.7 |
| 后端 | Nitro (Node.js) |

## 📊 支持市场

- 港股 (HK)
- A股 (CN) 
- 美股 (US)

## 📈 K线周期

- 日K、周K、月K

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动前端
npm run dev

# 启动后端 (另一个终端)
npm run dev:server
```

                            ↓

│  /api/stocks/news → 获取新闻                            │
│  /api/analysis → AI 分析                               │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐

## 🎯 核心分析逻辑

### 1. 当日走势分析

```
输入: 当日K线 + 当日新闻
    ↓
思维链:
  [1] 数据收集 → [2] 价格分析 → [3] 因素提取 → [4] 权重计算 → [5] 结论
    ↓
输出: { factors: [{name, weight, reason}], conclusion, thinkingChain }
```

### 2. 区间涨跌分析

```
输入: 日期区间K线 + 新闻
    ↓
思维链:
  [1] 数据收集 → [2] 趋势分析 → [3] 因素分析 → [4] 归因 → [5] 结论
    ↓
输出: { factors: 归因权重, conclusion, thinkingChain }
```

### 3. 未来走势预测

```
输入: 历史K线 + 新闻
    ↓
思维链:
  [1] 历史分析 → [2] 模式识别 → [3] 趋势外推 → [4] 新闻影响 → [5] 综合预测
    ↓
输出: { predictions: [{horizon, direction, confidence, reason}], thinkingChain }
```
