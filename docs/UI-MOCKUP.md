## 总结：界面特点

1. **左侧边栏** - 股票选择、参数设置、分析入口
2. **右侧主区域** - K线图可视化展示
3. **分析面板** - 折叠式多维度分析
4. **思维链** - 实时展示AI分析进度
5. **影响因素权重** - 可视化展示各因素贡献
6. **结论置信度** - 量化分析可靠性
